About AutoRxGen

AutoRxGen addresses two main issues:

  • The hassle of renewing chronic scripts for certain S2-S4 medications.
  • The need to physically visit a doctor for certain S2-S4 medications, if you already know what you need.

Features

Placeholder Image

Our application offers two main features:

Script Renewal

Users can upload a legitimate expired script, which is verified by the app. The script is then renewed with today's date and a new doctor's name. Alternatively, each script can appear on a doctor's dashboard for approval or rejection.

First-time Scripts

For certain low-schedule medications like topical minoxidil, users can fill out a questionnaire. Eligibility is determined based on criteria set by a doctor, with approvals or rejections based on the users' answers. All questions are created by a doctor.
